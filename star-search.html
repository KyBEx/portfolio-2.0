Wrate Your Wrath Dot Com

Star Search is another full-stack application that utilizes the MERN stack. It is a pair coded application I built with a colleague at V School. The idea was born out of our mutual interest in movies and the desire to feature an API heavily in the project.

The idea is that the site acts as an online resume for aspiring or established thespians. A user may sign up for the site and create a profile that showcases their bio and movie credits. The movie credits are pulled from TMDB, a RESTful API.

Once a user has signed up, their information is stored in a custom MongoDB database that is accessible from the search feature on the main page. This information can be accessed by anyone--a fan, a casting director, anyone who accesses the main page. Currently, only one record, Bruce Willis, is in this custom database.

Star Search uses Express for server routing and leverages token authentication via JWTs to ensure that users only have access to their own data. It uses Node / Express to handle server requests, and uses the React library to handle dynamic data updates on the front-end.
